🔍 WHAT IS SQL INJECTION?

SQL Injection happens when user input is directly inserted into SQL queries without proper validation. An attacker can break the query logic to bypass authentication or access unauthorized data.

🔎 HOW IT WORKS HERE

Normal Query:

SELECT * FROM users WHERE username='jsmith'
AND password='pass123'

Injected Query: If username is: jsmith' OR '1'='1

The query becomes:

SELECT * FROM users WHERE username='jsmith' OR '1'='1'
AND password='anything'

Since '1'='1' is ALWAYS TRUE, it bypasses the password check!
