Mission Briefing

Intel Found

During reconnaissance, you discovered a vulnerable network diagnostic tool. The server-side script takes the user-provided hostname and executes a system-level ping command using subprocess.check_output(command, shell=True).

The Vulnerability

OS Command Injection. Because the application uses shell=True and does not sanitize input, an attacker can use shell metacharacters (like ;, &, |) to append additional commands. These commands will execute with the same privileges as the web application.

Your Objective

The system's secret flag is stored in a file called /flag. Use command injection techniques to leak the contents of this file.

Payload Arsenal

8.8.8.8 ; ls /
8.8.8.8 && cat /flag
8.8.8.8 | cat /etc/passwd

Note: The test environment is Linux-based.
