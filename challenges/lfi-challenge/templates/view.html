Mission Intel

01 Introduction

This application uses a page parameter to fetch and display local files. This is a common pattern for documentation sites, but it's highly dangerous if not sanitized.

02 The Vulnerability

Local File Inclusion (LFI) occurs when an application includes a file from the local file system without proper validation. Attackers can use ../ sequences to navigate outside the intended directory.

03 Tactical Execution

Try to read sensitive system files. Use path traversal in the address bar: ../../../../etc/flag.txt. The ../ jumps up one directory level.

04 Flag Acquisition

Once you successfully access /etc/flag.txt, the system will detect the unauthorized read and reveal the system flag via the compromise modal.
